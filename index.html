Mini- Autonomous vehicle (2022) :
Using Python, openCV, Tensorflow, DonkeyCar Sims, and OnShape CAD software, I worked alongside a team of 2 others to design and print a chassis and camera mount for a small autonomous vehicle, which we trained using sim and real life data.

Pomodoro with Spotify Integration (ongoing) : Using the Spotify API, take in the user’s account info to grab their currently playing album covers and songs, which will translate into major colors that create the theme of the pomodoro timer/task manager.

Plant classification :
I built a plant classification model based on 105 different plants (csv data) and assembled an image database using a Selenium webscraper. Using this database, I trained a Sequential model. This classification model is fairly accurate for flower classification.

Water reminder bottle attachment :
Using an Arduino, load cell, 3D printed parts, and an HX711 ADC my group and I designed an built a device that beeps when you should drink water, continuing to flash an LED until a certain volume of water is drunk. Engineering class project.
